# Hangman - Beginner

Welcome to Hangman. My first of many projects that I am going to create and showcase here on my Portfolio. 
Follow along and enjoy the fun games in between!

## How it works:

1. Clone or download the files into a designated folder on your local device. 
2. Ensure that you download the "Words" document and it's content as well OR ensure that the "Hangman_wordbank" file is stored in a "Word" folder located in the same place as the main.py folder. 
3. Run the code in your terminal or Python IDE of choice (PyCharm, VSCode, etc.).
4. Once up and running you can start by guessing a letter!
5. At the end of each round you will be prompted to continue the game or not. If you choose not to, the program will end and you need to run it again inorder to play again.</p>
